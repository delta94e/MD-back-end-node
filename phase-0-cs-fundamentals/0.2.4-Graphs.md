## 1. Graph Là Gì

Graph G = (V, E) — tập đỉnh V (vertices) và tập cạnh E (edges).

```
Undirected:           Directed (DiGraph):       Weighted:
  A --- B               A ──▶ B                  A ──5── B
  |   / |               │     │                  |      / |
  |  /  |               ▼     ▼                  3    2   4
  | /   |               C ──▶ D                  |  /     |
  C --- D                                        C ──1── D

Terminology:
  - Degree: số cạnh nối với node (undirected)
  - In-degree / Out-degree: trong directed graph
  - Path: chuỗi edges kết nối 2 nodes
  - Cycle: path bắt đầu và kết thúc tại cùng node
  - Connected: mọi node đều reachable từ mọi node khác
  - DAG: Directed Acyclic Graph (có hướng, không có cycle)
```

## 2. Biểu Diễn Graph

### 2.1 Adjacency List (phổ biến nhất)

```javascript
// Undirected graph
const graph = {
  A: ["B", "C"],
  B: ["A", "C", "D"],
  C: ["A", "B", "D"],
  D: ["B", "C"],
};

// Hoặc dùng Map (linh hoạt hơn)
const g = new Map();
g.set("A", ["B", "C"]);
g.set("B", ["A", "C", "D"]);

// Weighted graph
const weighted = {
  A: [
    ["B", 5],
    ["C", 3],
  ],
  B: [
    ["A", 5],
    ["C", 2],
    ["D", 4],
  ],
};
```

### 2.2 Adjacency Matrix

```
      A  B  C  D
  A [ 0, 1, 1, 0 ]
  B [ 1, 0, 1, 1 ]
  C [ 1, 1, 0, 1 ]
  D [ 0, 1, 1, 0 ]

  matrix[i][j] = 1 nếu có cạnh i→j, 0 nếu không
  Weighted: matrix[i][j] = weight thay vì 1
```

### 2.3 So sánh

|                   | Adjacency List              | Adjacency Matrix      |
| ----------------- | --------------------------- | --------------------- |
| Space             | O(V + E)                    | O(V²)                 |
| Add edge          | O(1)                        | O(1)                  |
| Check edge        | O(degree)                   | **O(1)**              |
| Iterate neighbors | **O(degree)**               | O(V)                  |
| Best for          | **Sparse** graphs (E << V²) | Dense graphs (E ≈ V²) |
| Phỏng vấn         | ✅ Hầu hết dùng             | Ít dùng               |

## 3. DFS — Depth-First Search

"Đi sâu hết mức có thể trước khi quay lại" — dùng Stack (hoặc recursion = implicit stack).

```
Graph:  A — B — E
        |   |
        C — D

DFS từ A (thứ tự phụ thuộc implementation):
  A → B → D → C → (quay lại B) → E

Stack trace:
  Push A    [A]
  Pop A     [] → visit A, push neighbors [B, C]
  Pop C     [B] → visit C, push D → [B, D]
  Pop D     [B] → visit D, push B (đã visit) → [B]
  Pop B     [] → visit B, push E → [E]
  Pop E     [] → visit E → DONE
```

```javascript
// Recursive DFS
function dfs(graph, node, visited = new Set()) {
  if (visited.has(node)) return;
  visited.add(node);
  console.log(node); // process
  for (const neighbor of graph[node]) {
    dfs(graph, neighbor, visited);
  }
}

// Iterative DFS (dùng explicit stack)
function dfsIterative(graph, start) {
  const visited = new Set();
  const stack = [start];
  while (stack.length) {
    const node = stack.pop();
    if (visited.has(node)) continue;
    visited.add(node);
    console.log(node);
    for (const neighbor of graph[node]) {
      if (!visited.has(neighbor)) stack.push(neighbor);
    }
  }
}
```

**DFS applications**: Cycle detection, topological sort, connected components, pathfinding (maze), backtracking.

## 4. BFS — Breadth-First Search

"Duyệt theo tầng" — dùng Queue. **Tìm shortest path** trong unweighted graph.

```javascript
function bfs(graph, start) {
  const visited = new Set([start]);
  const queue = [start];
  const distance = { [start]: 0 };

  while (queue.length) {
    const node = queue.shift();
    for (const neighbor of graph[node]) {
      if (!visited.has(neighbor)) {
        visited.add(neighbor);
        queue.push(neighbor);
        distance[neighbor] = distance[node] + 1;
      }
    }
  }
  return distance;
}
// BFS đảm bảo shortest path trong UNWEIGHTED graph!
```

### DFS vs BFS

|                | DFS                                            | BFS                                          |
| -------------- | ---------------------------------------------- | -------------------------------------------- |
| Data structure | Stack / Recursion                              | Queue                                        |
| Shortest path  | ❌ Không đảm bảo                               | ✅ Trong unweighted graph                    |
| Space          | O(depth)                                       | O(width of level)                            |
| Best for       | Pathfinding, cycle detection, topological sort | Shortest path, level-order, nearest neighbor |
| Grid problems  | OK                                             | ✅ Preferred                                 |

## 5. Shortest Path Algorithms

### 5.1 Dijkstra — Single source, non-negative weights

```javascript
function dijkstra(graph, start) {
  const dist = {};
  const visited = new Set();
  // graph[node] = [[neighbor, weight], ...]

  // Initialize
  for (const node in graph) dist[node] = Infinity;
  dist[start] = 0;

  // Priority Queue (simplified with array — use real MinHeap in production)
  const pq = [[0, start]]; // [distance, node]

  while (pq.length) {
    pq.sort((a, b) => a[0] - b[0]); // MinHeap simulation
    const [d, u] = pq.shift();

    if (visited.has(u)) continue;
    visited.add(u);

    for (const [v, weight] of graph[u]) {
      const newDist = dist[u] + weight;
      if (newDist < dist[v]) {
        dist[v] = newDist;
        pq.push([newDist, v]);
      }
    }
  }
  return dist;
}
// Time: O((V + E) log V) with MinHeap
// KHÔNG hoạt động với negative weights!
```

### 5.2 Bellman-Ford — Handles negative weights

```javascript
function bellmanFord(edges, V, start) {
  const dist = new Array(V).fill(Infinity);
  dist[start] = 0;

  // Relax tất cả edges V-1 lần
  for (let i = 0; i < V - 1; i++) {
    for (const [u, v, w] of edges) {
      if (dist[u] !== Infinity && dist[u] + w < dist[v]) {
        dist[v] = dist[u] + w;
      }
    }
  }

  // Check negative cycle (lần thứ V)
  for (const [u, v, w] of edges) {
    if (dist[u] !== Infinity && dist[u] + w < dist[v]) {
      throw new Error("Negative cycle detected!");
    }
  }
  return dist;
}
// Time: O(V × E) — chậm hơn Dijkstra nhưng handle negative weights
```

### 5.3 So sánh Shortest Path

| Algorithm          | Negative weights | Negative cycles | Time         | Use case                |
| ------------------ | :--------------: | :-------------: | ------------ | ----------------------- |
| **BFS**            |        ❌        |       ❌        | O(V+E)       | Unweighted graph        |
| **Dijkstra**       |        ❌        |       ❌        | O((V+E)logV) | Non-negative weighted   |
| **Bellman-Ford**   |        ✅        |     Detect      | O(V×E)       | Negative weights        |
| **Floyd-Warshall** |        ✅        |     Detect      | O(V³)        | All-pairs shortest path |

## 6. Topological Sort

**Chỉ cho DAG** (Directed Acyclic Graph). Sắp xếp nodes sao cho mọi edge u→v thì u đứng trước v.

```
Use case: task scheduling, build dependencies, course prerequisites

  Course dependencies:
    Calculus → Linear Algebra → Machine Learning
    Calculus → Statistics → Machine Learning
    Python → Machine Learning

  Topological order: Python, Calculus, Statistics, Linear Algebra, ML
  (nhiều valid orderings)
```

```javascript
// Kahn's Algorithm (BFS-based)
function topologicalSort(graph, V) {
  const inDegree = new Array(V).fill(0);
  for (let u = 0; u < V; u++) {
    for (const v of graph[u]) inDegree[v]++;
  }

  const queue = [];
  for (let i = 0; i < V; i++) {
    if (inDegree[i] === 0) queue.push(i);
  }

  const result = [];
  while (queue.length) {
    const u = queue.shift();
    result.push(u);
    for (const v of graph[u]) {
      inDegree[v]--;
      if (inDegree[v] === 0) queue.push(v);
    }
  }

  if (result.length !== V) throw new Error("Cycle detected!");
  return result;
}
```

## 7. Union-Find (Disjoint Set)

Quản lý **tập hợp rời rạc** — hỗ trợ `union(a, b)` và `find(a)` gần O(1).

```javascript
class UnionFind {
  constructor(n) {
    this.parent = Array.from({ length: n }, (_, i) => i);
    this.rank = new Array(n).fill(0);
    this.count = n; // số components
  }

  find(x) {
    if (this.parent[x] !== x) {
      this.parent[x] = this.find(this.parent[x]); // PATH COMPRESSION
    }
    return this.parent[x];
  }

  union(x, y) {
    const px = this.find(x),
      py = this.find(y);
    if (px === py) return false; // đã cùng set
    // UNION BY RANK
    if (this.rank[px] < this.rank[py]) this.parent[px] = py;
    else if (this.rank[px] > this.rank[py]) this.parent[py] = px;
    else {
      this.parent[py] = px;
      this.rank[px]++;
    }
    this.count--;
    return true;
  }

  connected(x, y) {
    return this.find(x) === this.find(y);
  }
}
// With both optimizations: near O(α(n)) per operation
// α(n) = inverse Ackermann function ≈ O(1) in practice
```

**Use cases**: Connected components, Kruskal's MST, cycle detection (undirected), network connectivity.

## 8. Advanced: MST & SCC

### Minimum Spanning Tree (Kruskal)

```javascript
function kruskal(edges, V) {
  edges.sort((a, b) => a[2] - b[2]); // sort by weight
  const uf = new UnionFind(V);
  const mst = [];
  for (const [u, v, w] of edges) {
    if (uf.union(u, v)) {
      // no cycle
      mst.push([u, v, w]);
      if (mst.length === V - 1) break;
    }
  }
  return mst;
}
// Time: O(E log E) — dominated by sorting
```

**Grid as Graph** — Pattern cực phổ biến:

```javascript
// Grid = implicit graph. Mỗi cell = node. 4 neighbors = edges.
const DIRS = [
  [0, 1],
  [0, -1],
  [1, 0],
  [-1, 0],
]; // right, left, down, up

function numIslands(grid) {
  let count = 0;
  const rows = grid.length,
    cols = grid[0].length;

  function dfs(r, c) {
    if (r < 0 || r >= rows || c < 0 || c >= cols) return;
    if (grid[r][c] !== "1") return;
    grid[r][c] = "0"; // mark visited
    for (const [dr, dc] of DIRS) dfs(r + dr, c + dc);
  }

  for (let r = 0; r < rows; r++) {
    for (let c = 0; c < cols; c++) {
      if (grid[r][c] === "1") {
        count++;
        dfs(r, c);
      }
    }
  }
  return count;
}
```
